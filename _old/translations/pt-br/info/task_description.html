Os números romanos vêm do antigo sistema de numeração da Roma. Eles são baseados na combinação de letras específicas do alfabeto que são combinadas para representar a soma (ou, em alguns casos, a subtraçao) de seus valores. Os primeiros dez números romanos são:

I, II, III, IV, V, VI, VII, VIII, IX e X.

O sistema dos números Romanos é baseado no decimal, mas não diretamente posicional e não inclui o zero. Os números romanos são baseados na combinação destes sete símbolos:

  • Símbolo Valor
  • I 1 (unus)
  • V 5 (quinque)
  • X 10 (decem)
  • L 50 (quinquaginta)
  • C 100 (centum)
  • D 500 (quingenti)
  • M 1.000 (mille)

Mais informações sobre os números Romanos podem ser encontradas no artigo da Wikipedia.

Para essa missão, você deve retornar um número romano usando o número inteiro especificado que vai de 1 a 3999.

Entrada: Um número inteiro.

Saída: O número romano em string.

Exemplo: 

checkio(6) == 'VI'
checkio(76) == 'LXXVI'
checkio(13) == 'XIII'
checkio(44) == 'XLIV'
checkio(3999) == 'MMMCMXCIX'

Onde é usado: Este é uma missão educacional que permite explorar diferentes sistemas de numeração. Desde que sistemas de numeração são frequentemente utilizados na tipografia, podem ser usados na criação de textos.
